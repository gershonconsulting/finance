HOW USERS EXPORT:
=================

1. Click green "Export to Google Sheets" button
2. CSV file downloads automatically
3. Open Google Sheets (sheets.google.com)
4. File → Import → Upload
5. Select the downloaded CSV
6. Click "Import data"
7. Done! Data is now in Google Sheets
