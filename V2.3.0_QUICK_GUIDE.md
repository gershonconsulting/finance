## 🔧 Troubleshooting

### **Issue: Health endpoint shows old version**
**Solution:**
1. In GenSpark, click "Delete Deployment"
2. Wait 1 minute
3. Click "Deploy" again
4. Wait 2-3 minutes
5. Check health endpoint again

---

### **Issue: Sheets Links still show sandbox URLs**
**Solution:**
1. Hard refresh browser: **Ctrl+Shift+R** (Windows) or **Cmd+Shift+R** (Mac)
2. If still wrong: Clear browser cache completely
3. If still wrong: Try incognito mode
4. If still wrong: GenSpark didn't deploy v2.3.0 properly (delete and redeploy)

---

### **Issue: Clients tab still shows error**
**Solution:**
1. Verify health endpoint shows `"version": "2.3.0"`
2. If yes: Open browser DevTools (F12) → Network tab → Refresh
3. Find request to `/api/demo/clients-awaiting-payment`
4. Check response has `totalOutstanding` field
5. If no: GenSpark deployed wrong version (delete and redeploy)
