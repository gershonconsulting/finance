Clear Settings & Reset

Fix "invalid_client" authentication errors

Error: Token exchange failed: {"error":"invalid_client"}

This means your Client Secret is incorrect or missing.

What This Does

  • Clears saved Client ID and Client Secret
  • Removes authentication session
  • Loads working default credentials (optional)
  • Redirects to dashboard Settings tab

Tip: After clearing, use "Load Default Configuration" in Settings to use pre-configured, working credentials.
